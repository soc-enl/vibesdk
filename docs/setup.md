# Setup

This project is now a static Cloudflare Pages app.

## Requirements

- Node.js 18+
- npm
- Cloudflare account with Pages access

## Install

```bash
npm install
```

## Run locally

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

```bash
npm run deploy
```

The deploy command publishes `dist` to the Pages project `vibesdk-app`.
